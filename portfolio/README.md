# [Start Bootstrap](http://startbootstrap.com/) - [Portfolio Item Template](http://startbootstrap.com/template-overviews/portfolio-item/)

**[Portfolio Item](http://startbootstrap.com/template-overviews/portfolio-item/)** is a sleek, individual portfolio item template built using the [Bootstrap](http://getbootstrap.com/) framework. It’s designed to help developers and designers quickly showcase their projects in a visually appealing way. This template is part of the **Start Bootstrap** collection, known for providing high-quality open-source templates built on Bootstrap.

---

## Getting Started  

To start using this template, follow one of the options below:  

1. **Download the Latest Release:**  
   - Head to the [Portfolio Item Template Page](http://startbootstrap.com/template-overviews/portoflio-item/) and download the latest version directly.  

2. **Clone the Repository:**  
   - Use the following command to clone the template repository:
     ```bash
     git clone https://github.com/BlackrockDigital/startbootstrap-portoflio-item.git
     ```

3. **Fork the Repository:**  
   - Click the **Fork** button on the GitHub repository page to create your own copy, allowing you to modify and maintain it under your GitHub account.

## Bugs and Issues  

Encountered a bug or an issue with the template? No worries! Here’s how you can report it:  

- **Open a New Issue:**  
  - Go to the [GitHub Issues Page](https://github.com/BlackrockDigital/startbootstrap-portoflio-item/issues) and create a new issue to describe the problem in detail.

- **Leave a Comment on the Template Overview Page:**  
  - Alternatively, you can leave feedback on the [Portfolio Item Overview Page](http://startbootstrap.com/template-overviews/portoflio-item/).

Providing clear information about the issue helps us resolve it quickly!
